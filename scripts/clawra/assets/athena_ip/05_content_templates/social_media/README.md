# Athena 社交媒体内容模板

## 设计理念
- **技术叙事**：将技术概念包装成引人入胜的故事
- **视觉冲击**：使用漫威电影风格的视觉元素
- **认知分层**：三体式叙事结构，从现象到本质
- **硅基共生**：突出AI与人类智慧的协作价值

## 模板分类

## 视觉规范

### 1. 图像模板
| 模板类型 | 尺寸 | 主要元素 | 配色方案 |
|----------|------|----------|----------|
| **Twitter卡片** | 1200x675px | 智慧之眼 + 数据流 | 科技蓝 + 白底 |
| **LinkedIn文章** | 1200x627px | 几何数据体 + 文字 | 深空紫渐变 |
| **Instagram帖子** | 1080x1080px | 全息助手 + 动态效果 | 霓虹科技感 |

### 2. 视频模板
| 时长 | 结构 | 节奏 | 音乐风格 |
|------|------|------|----------|
| **15秒短视** | 问题→解决→价值 | 快节奏 | 电子合成音 |
| **1分钟教程** | 理论→演示→总结 | 中节奏 | 科技氛围音 |
| **3分钟深度** | 背景→分析→展望 | 慢节奏 | 交响乐+电子 |

### 3. 文案风格矩阵
| 平台 | 语气 | 技术深度 | 情感强度 | 互动期望 |
|------|------|----------|----------|----------|
| **Twitter** | 精炼有力 | 中等 | 较高 | 高转发，高讨论 |
| **LinkedIn** | 专业深度 | 高 | 中等 | 专业认可，连接 |
| **GitHub** | 技术严谨 | 很高 | 较低 | 代码贡献，问题反馈 |
| **Reddit** | 社区友好 | 可变 | 较高 | 深度讨论，投票 |

## 使用指南

### 1. 内容创作流程
```
1. 确定目标用户和平台
2. 选择合适的模板类型
3. 填入具体内容（保持三体叙事结构）
4. 添加品牌视觉元素
5. 质量检查：技术准确性 + 情感连接
6. 发布和监测效果
```

### 2. 三体叙事结构应用
- **第一层**：现象/问题（引发共鸣）
- **第二层**：分析/解决方案（技术深度）
- **第三层**：影响/展望（宏大视角）

### 3. 品牌元素集成
- **Logo位置**：右上角或左下角，不干扰内容
- **色彩使用**：主色调占比60%，辅助色30%，中性色10%
- **动态效果**：适度使用，不分散注意力

## 质量检查清单
- [ ] 技术内容100%准确
