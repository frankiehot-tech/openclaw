# Athena 品牌色彩规范

## 1. 核心色彩体系

### 1.1 主色调 - 科技智慧蓝
- **色值**: `#4A90E2`
- **RGB**: `74, 144, 226`
- **CMYK**: `67, 36, 0, 11`
- **HSL**: `214°, 74%, 59%`
- **应用场景**: 品牌标识、主按钮、重要信息强调

**设计意义**:
- 代表科技与智慧的结合
- 体现AI的专业性和可靠性
- 视觉上冷静而不冷漠，理性而富有情感

### 1.2 数据流动绿
- **色值**: `#00D4AA`
- **RGB**: `0, 212, 170`
- **CMYK**: `100, 0, 20, 17`
- **HSL**: `168°, 100%, 42%`
- **应用场景**: 数据流动效果、成功状态、活力元素

**设计意义**:
- 象征数据的流动和生命
- 代表成长、进化和正能量
- 在科技蓝基础上增加生机感

## 4. 色彩组合规范

### 4.1 标准组合（推荐）
- **主组合**: `#4A90E2` + `#1A1A2E` + `#F5F7FA`
- **活力组合**: `#00D4AA` + `#7B61FF` + `#1A1A2E`
- **警示组合**: `#FF6B35` + `#1A1A2E` + `#F5F7FA`

### 4.2 使用比例
- **主色调占比**: 60-70%
- **辅助色调占比**: 20-30%
- **中性色调占比**: 10-20%
- **强调色调占比**: 5-10%

### 4.3 可访问性对比度要求
- **AA级标准** (最小对比度):
  - 正常文本: 4.5:1
  - 大文本: 3:1
- **AAA级标准** (增强对比度):
  - 正常文本: 7:1
  - 大文本: 4.5:1

## 6. 场景应用示例

### 6.1 技术文档界面
```css
/* 深色模式 */
.dark-mode {
  background: var(--athena-dark-gray);
  color: var(--athena-light-gray);
  accent-color: var(--athena-primary-blue);
}

/* 代码块 */
.code-block {
  background: #2D2D44;
  border-left: 4px solid var(--athena-data-green);
}
```

### 6.2 数据可视化
```css
/* 数据图表 */
.chart-line {
  stroke: var(--athena-primary-blue);
}

.chart-area {
  fill: var(--athena-data-green);
  opacity: 0.3;
}

/* 数据点 */
.data-point {
  fill: var(--athena-energy-orange);
}
```

### 6.3 用户界面组件
```css
/* 按钮 */
.btn-primary {
  background: linear-gradient(135deg, var(--athena-primary-blue), var(--athena-space-purple));
  color: white;
}

.btn-secondary {
  background: var(--athena-dark-gray);
  border: 2px solid var(--athena-data-green);
  color: var(--athena-data-green);
}
```
