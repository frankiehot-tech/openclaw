# UI/UX Pro Max

## 触发条件
- 用户提到 UI、UX、界面设计、用户体验、CSS 样式、响应式
- 前端组件开发或重构

## 执行步骤

### 1. 设计审查
- 检查色彩对比度 (WCAG AA/AAA)
- 检查字体层级一致性
- 检查间距系统 (4px/8px grid)
- 检查响应式断点

### 2. 可访问性
- 键盘导航支持
- ARIA 标签完整性
- 屏幕阅读器兼容

### 3. 输出格式
```
发现问题清单 + 修复建议 + 优先级 (P0/P1/P2)
```
