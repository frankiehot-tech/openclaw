## 首轮验收任务清单

### 1. 导航类任务（最多5个）
1. **返回桌面并打开指定 App**
   - 任务描述：从任意界面返回主屏幕，找到并打开指定应用（如"设置"）
   - 复杂度：中等（需要识别图标+文字）
   - 风险等级：低

2. **找到设置并进入 Wi-Fi 页面**
   - 任务描述：打开设置应用，找到并进入 Wi-Fi 设置页面
   - 复杂度：中等（需要多层导航）
   - 风险等级：低

3. **返回上一级菜单**
   - 任务描述：从子页面返回到父级菜单
   - 复杂度：低（单一动作）
   - 风险等级：低

4. **打开通知中心**
   - 任务描述：从主屏幕下拉打开通知中心
   - 复杂度：低（手势操作）
   - 风险等级：低

5. **切换应用**
   - 任务描述：从当前应用切换到另一个已打开的应用
   - 复杂度：中等（需要识别最近应用界面）
   - 风险等级：低

### 2. 查找类任务（最多5个）
1. **查找特定设置项**
   - 任务描述：在设置中查找特定项目（如"关于手机"）
   - 复杂度：中等（需要滚动+识别）
   - 风险等级：低

2. **查找联系人**
   - 任务描述：在通讯录中查找指定联系人
   - 复杂度：中等（需要搜索或滚动）
   - 风险等级：低

3. **查找应用商店中的 App**
   - 任务描述：在应用商店中搜索并找到指定应用
   - 复杂度：高（需要输入+识别）
   - 风险等级：中

4. **查找文件管理器中的文件**
   - 任务描述：在文件管理器中找到指定文件
   - 复杂度：中等（需要目录导航）
   - 风险等级：低

5. **查找日历中的事件**
   - 任务描述：在日历中找到今天的事件
   - 复杂度：中等（需要日期识别）
   - 风险等级：低

### 3. 整理类任务（最多5个）
1. **整理桌面第一页**
   - 任务描述：将桌面第一页的图标按类型整理
   - 复杂度：高（需要多步拖拽操作）
   - 风险等级：中

2. **清理通知**
   - 任务描述：清除所有通知
   - 复杂度：低（单一动作）
   - 风险等级：低

3. **关闭后台应用**
   - 任务描述：清理所有后台运行的应用
   - 复杂度：中等（需要手势+识别）
   - 风险等级：低

4. **整理相册**
   - 任务描述：将最近的照片移动到指定相册
   - 复杂度：高（需要多步选择操作）
   - 风险等级：中

5. **整理下载文件夹**
   - 任务描述：将下载文件夹中的文件分类
   - 复杂度：高（需要识别文件类型）
   - 风险等级：中

## 3个优先测试任务

### 任务1：返回桌面并打开指定 App
**任务描述**：
1. 从任意当前界面返回主屏幕
2. 在主屏幕上找到"设置"应用图标
3. 点击打开设置应用

**通过条件**：
1. ✅ 成功返回主屏幕（屏幕显示桌面图标）
2. ✅ 正确识别"设置"应用图标（OCR识别到"设置"文字）
3. ✅ 成功打开设置应用（屏幕显示设置界面）
4. ✅ 总步骤数 ≤ 5步
5. ✅ 执行时间 ≤ 30秒

**失败类型**：
- 导航失败：无法返回主屏幕
- 识别错误：找不到或误识别"设置"图标
- 执行错误：点击位置错误
- 超时：超过30秒未完成

### 任务2：找到设置并进入 Wi-Fi 页面
**任务描述**：
1. 在设置应用中找到"连接"或"网络"相关选项
2. 点击进入网络设置页面
3. 找到并点击"Wi-Fi"选项
4. 进入Wi-Fi设置页面

**通过条件**：
1. ✅ 成功进入设置应用
2. ✅ 正确找到网络相关选项（OCR识别到"连接"、"网络"或类似文字）
3. ✅ 成功进入Wi-Fi页面（屏幕显示Wi-Fi开关和网络列表）
4. ✅ 总步骤数 ≤ 8步
5. ✅ 执行时间 ≤ 45秒

**失败类型**：
- 导航失败：在设置中迷路
- 识别错误：找不到Wi-Fi相关选项
- 层级错误：进入错误的子页面
- 超时：超过45秒未完成

### 任务3：整理桌面第一页
**任务描述**：
1. 确保在主屏幕第一页
2. 识别桌面上的应用图标
3. 将工具类应用移动到屏幕右侧区域
4. 将娱乐类应用移动到屏幕左侧区域
5. 整理后图标布局有明显分区

**通过条件**：
1. ✅ 正确识别至少5个应用图标类型
2. ✅ 成功执行至少3次拖拽移动操作
3. ✅ 整理后图标布局有明显分区效果
4. ✅ 未误删除或误操作其他图标
5. ✅ 总步骤数 ≤ 10步
6. ✅ 执行时间 ≤ 60秒

**失败类型**：
- 识别错误：无法正确分类应用类型
- 操作错误：拖拽失败或误操作
- 布局错误：整理后无明显分区
- 超时：超过60秒未完成
