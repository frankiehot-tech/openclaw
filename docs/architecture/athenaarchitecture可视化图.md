## 🎯 核心组件功能矩阵

### 用户交互层组件

| 组件 | 图标 | 主要功能 | 技术栈 | 部署方式 |
|------|------|----------|--------|----------|
| **Web Desktop** | 🖥️ | 统一控制面、任务监控 | React + TS | 独立部署 |
| **VS Code** | 💻 | IDE 集成开发环境 | VS Code 扩展 | 本地安装 |
| **CLI 工具** | ⌨️ | 命令行接口、自动化 | Python + Click | 本地/服务器 |
| **移动端界面** | 📱 | 移动访问、通知推送 | React Native | 移动端部署 |

### 编排调度层组件

| 组件 | 图标 | 主要功能 | 技术栈 | 关键特性 |
|------|------|----------|--------|----------|
| **AI Plan 队列** | 📋 | 任务编排、优先级调度 | Redis + Celery | 重试机制 |
| **Codex 编排** | 🧠 | AI 任务分解优化 | OpenAI API | 智能分配 |
| **LangGraph** | 📊 | 状态机驱动协作 | LangGraph 框架 | 类型安全 |
| **OpenCode CLI** | ⚡ | 工作流执行引擎 | 自定义 CLI | 高性能 |

### 多 Agent 协作层组件

| Agent | 图标 | 角色定位 | 核心能力 | 输出产物 |
|-------|------|----------|----------|----------|
| **Orchestrator** | 🎯 | 系统总协调员 | 任务分配、质量门禁 | 工作流状态 |
| **Researcher** | 🔍 | 信息研究员 | 关键词挖掘、竞品分析 | 研究数据 |
| **Writer** | ✍️ | 内容创作者 | 大纲生成、正文撰写 | 内容初稿 |
| **Validator** | ✅ | 质量审核员 | GEO 评分、事实核查 | 质量报告 |
| **Publisher** | 🚀 | 发布管理员 | 多平台发布、数据回流 | 发布状态 |

### 工具服务层组件

| 工具 | 图标 | 功能描述 | 集成方式 | 缓存策略 |
|------|------|----------|----------|----------|
| **Athena Tool Kit** | 🛠️ | 统一工具封装层 | Python 装饰器 | Redis 缓存 |
| **serpapi-cli** | 🔍 | 搜索引擎结果获取 | REST API | 1小时缓存 |
| **readability** | 📖 | 内容可读性分析 | CLI 工具 | 无缓存 |
| **ollama-api** | 🤖 | 本地 AI 模型调用 | HTTP API | 会话缓存 |

### 数据存储层组件

| 存储 | 图标 | 数据类型 | 技术方案 | 备份策略 |
|------|------|----------|----------|----------|
| **PostgreSQL** | 🗄️ | 业务数据、用户数据 | 主从复制 | WAL 归档 |
| **Redis 集群** | 🔥 | 缓存数据、会话数据 | 集群模式 | AOF + RDB |
| **对象存储** | 💾 | 文件存储、日志文件 | S3 兼容 | 多副本 |
| **监控数据库** | 📊 | 监控指标、性能数据 | TimescaleDB | 自动清理 |

### 监控运维层组件

| 组件 | 图标 | 监控范围 | 告警方式 | 可视化工具 |
|------|------|----------|----------|------------|
| **Prometheus** | 📊 | 系统指标、业务指标 | 多通道告警 | Grafana |
| **Grafana** | 📈 | 性能趋势、实时状态 | 仪表板展示 | 自定义面板 |
| **ELK Stack** | 📋 | 日志分析、故障定位 | 搜索分析 | Kibana |
| **告警系统** | 🚨 | 异常检测、阈值告警 | 邮件/短信 | 自定义规则 |
