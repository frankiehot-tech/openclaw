分析 128k+ token 的约束，揭示了标准多头注意力 (MHA) 实现中显著的非线性扩展问题。

主要的退化并非发生在纯内存分配上，而是注意力矩阵的信噪比。随着序列长度 (N) 的增加，N 个键上的 softmax 函数趋于平坦，导致模型过度关注高频但语义无关的 token（即"干扰效应"）。

def calculate_attention_entropy(attn_weights):
    probs = torch.softmax(attn_weights, dim=-1)
    entropy = -torch.sum(probs * torch.log(probs + 1e-9), dim=-1)
    return entropy.mean().item()

为缓解此问题，建议迁移至稀疏注意力 (Sparse Attention) 或实施环形注意力 (Ring Attention) 架构，分布式部署于 GPU 集群中，以在超出当前限制时保持持续的连贯性。
